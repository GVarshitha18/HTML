Heading

  1. There are 6 heading tags
  2. h1 to h6
  3. h1 main title, h2 subtitle, h3 small title, ...

Tags

  1. There are 2 tags
  2. Paired tags
  3. unpaired tags
  4. In a paired tags we have both open and closing tags like 'P',NAV
  5. In a unpaired tag there will be only a open tag

Elements

  1. An Element consists of the tags and content
  2. By using p tag we will write the content in between the tags
Attributes
  1. Attribute are in the form of name="varsh", age="21", place="Atp"
  2. Attributes will give more information about an element
Paragraphs & lines breaks
  1. Paragraphs are written in 'p' tags
  2. Line breaks are written using the 'br' tags
